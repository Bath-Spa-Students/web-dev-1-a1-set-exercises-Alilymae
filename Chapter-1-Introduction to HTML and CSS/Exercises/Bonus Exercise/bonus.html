Basic HTML Tags


  1. !DOCTYPE


    This is the very first tag in an HTML document, it defines
    the document type.
    2. Example: <!DOCTYPE>

  2. HTML


    This tag defines the HTML Document.
    3. Example: <html>...</html>

  3. Head


    This tag contains the metadata/information for the document.
    4. Example: <head>...</head>

  4. Body


    This tag defines the documents body and contents.
    5. Example: <body>...</body>

  5. Paragraph


    This is the paragraph tag which is placed inside the body.
    6. Example: <p>...</p>

Anchor tag

Defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the
href attribute, which indicates the link's destination. 


Example: <a href="https://www.youtube.com/watch?v=dQw4w9WgXcQ">My Favorite Song</a>

Footer tag

This tag defines a footer for a document or section. 


Example: <footer>Copyright© All Rights Reserved</footer>

Division tag

This tag defines a division or a section in an HTML document. Also used as a container for HTML elements - which is then
styled with CSS or manipulated with JavaScript. This tag is easily styled by using the class or id attribute 


Example: 

                    <div class = "text1">

                        <p>Bob says Hello</p> 

                    </div>

Image tag

This tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are
linked to web pages. The <img> tag creates a holding space for the referenced image.

It has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed


Example: <img src="img/howl.jpg" alt="howls moving castle" height="200" width="400">


What is Cascading Stylesheets (CSS)?

CSS is the language we use to style an HTML document.
CSS describes how HTML elements should be displayed.

Inline Styling

Inline CSS styles an individual HTML element by adding a style attribute on the element's opening tag.

Internal Styling

Internal CSS styles an HTML page's content by adding an HTML <style> element in the <head> section of
the HTML document.

External Styling

External CSS styles one or more HTML pages' content by linking the HTML document to an external stylesheet.

CSS Colors

Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values. You can change the colors of
the background, text, border color, and etc.

  • Background-color
    - You can set the background color for HTML elements
  • Font color
    - Setting the color of the text
  • Border-color
    - Setting the color of borders
Example: Changing background color using color names 

                    <h1 style="background-color:Tomato;">Tomato</h1>
                    <h1 style="background-color:Orange;">Orange</h1>
                    <h1 style="background-color:DodgerBlue;">DodgerBlue</h1>
                    <h1 style="background-color:MediumSeaGreen;">MediumSeaGreen</h1>
                    <h1 style="background-color:Gray;">Gray</h1>
                    <h1 style="background-color:SlateBlue;">SlateBlue</h1>
                    <h1 style="background-color:Violet;">Violet</h1>
                    <h1 style="background-color:LightGray;">LightGray</h1>
                



Example: Changing font color using RGB 

                    <h1 style="color: rgb(201, 16, 16)">Tomato</h1>

CSS Font Properties

Styling the fonts of the text in the HTML file.

  • Normal - The text is shown normally
  • Italic - The text is shown in italics
  • Oblique - The text is "leaning" (oblique is very similar to italic, but less supported)
Example of font style property: 

                    <!DOCTYPE html>
                    <html>
                    <head>
                    <style>
                        p.normal {
                        font-style: normal;
                        }

                        p.italic {
                        font-style: italic;
                        }

                        p.oblique {
                        font-style: oblique;
                        }
                    </style>
                    </head>
                    <body>

                    <h1>The font-style property</h1>

                    <p class="normal">This is a paragraph in normal style.</p>
                    <p class="italic">This is a paragraph in italic style.</p>
                    <p class="oblique">This is a paragraph in oblique style.</p>

                    </body>>
                    </html>

Font Families

  1. Serif fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance.
  2. Sans-serif fonts have clean lines (no small strokes attached). They create a modern and minimalistic look.
  3. Monospace fonts - here all the letters have the same fixed width. They create a mechanical look.
  4. Cursive fonts imitate human handwriting.
  5. Fantasy fonts are decorative/playful fonts.
